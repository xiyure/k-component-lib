# 全局配置
`@ksware/ksw-ux` 提供了全局配置国际化的配置。
::: tip 提示
组件库目前仅支持中英双语切换
:::

```typescript
import kswUx from '@ksware/ksw-ux';

app.use(kswUx, {
  locale: 'zh',
});
```
# ConfigProvider
`@ksware/ksw-ux` 还提供了一个 `Vue` 组件 `GlobalProvider` 用于全局配置国际化的设置。

```html
<template>
  <k-global-provider :locale="locale">
    <app />
  </k-global-provider>
</template>
<script setup lang="ts">
import { ref } from 'vue';

const locale = ref('zh');
</script>
```
