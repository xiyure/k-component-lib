# 快速上手

ksw-ux 致力于提供给程序员愉悦的开发体验。

> 在开始之前，推荐先学习 Vue 3.x 和 ES2015，并正确安装和配置了 Node.js v16 或以上。官方指南假设你已了解关于 HTML、CSS 和 JavaScript 的中级知识，并且已经完全掌握了 Vue 3.x 的正确开发方式。如果你刚开始学习前端或者 Vue，将 UI 框架作为你的第一步可能不是最好的主意。
