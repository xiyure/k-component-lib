# Button 按钮

常用的操作按钮。

## 基础用法

注重业务区分, 按钮类型分别为:

- 普通按钮:
  > 默认状态, 常用用于不会产生数据的操作;
- 主要按钮:
  > 模块中一般只存在一个, 用于重要操作;
- 次级按钮
  > 模块中可以有多个, 用户产生数据的操作;
- 文字按钮
  > 一般用于链接, 或者表格中使用;
- icon按钮
  > 一般用于节省空间的情景。
